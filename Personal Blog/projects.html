My Android Unity Ninja Runner Game Project
I'm currently developing an exciting 2D/3D runner game for Android using Unity. This project aims to create an engaging and dynamic experience for players, combining fast-paced gameplay with a unique visual style. The game features a character running through various challenging environments, collecting items, and avoiding obstacles to reach the highest score.
The game mechanics include smooth controls for jumping and sliding, along with randomly generated levels that keep each playthrough fresh and unpredictable. I've implemented a scoring system that rewards players for collecting power-ups and performing special moves. Additionally, I’m working on integrating features like unlockable characters, different environments, and in-game achievements.
As I continue to develop the project, I’m focusing on optimizing the game for various Android devices, ensuring a smooth and enjoyable experience across different screen sizes and hardware capabilities. I look forward to sharing more updates as the project progresses!
